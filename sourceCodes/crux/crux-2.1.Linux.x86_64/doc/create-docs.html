create-docs

Usage:

crux create-docs [options] <tool name>

Description:

This command prints to standard output an HTML formatted version of the documentation for a specified Crux command.

Input:

  • <tool name> – Specifies the Crux tool to generate documentation for. If the value is 'list', then a list of available tools will be given. If the value is 'default-params', then a default parameter file will be given.

Output:

  • stdout – The command prints to standard output the HTML documentation for the specified Crux tool.

Options:

  • This command does not support any optional parameters.
